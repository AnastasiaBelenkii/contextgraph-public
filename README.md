# ContextGraph

An infinite canvas where every LLM message is a node you can wire to other nodes, giving you direct control over what context the AI sees. 

## Status

ContextGraph is experimental and evolving fast. Agent integration, paper import, and more are coming soon.

## Try it

[contextgraph.me](contextgraph.me) — enter your email and start building graphs.

## Roadmap

- [ ] Keyboard shortcuts and better graph navigation
- [ ] Semantic zoom and better node labeling
- [ ] Node hiding
- [ ] Paper/document import as graph nodes
- [ ] Linear chat mode with graph affordances
- [ ] Coding agent integration (bring your own agent)
- [ ] Global node search
